HEllO

    Список назв змінних для завдання:
  • - batterConsistency - консистенція тіста - жидкое
  • - patelniaTemperature - температура пательні(сковородки) - 140-150 oС
  • - cookingTime - час приготування - 3хв на одній стороні
  • - flipCount - кількість перевертань - 2
  • - ingredientList - список інгредієнтів - яйця, мука, молоко, сіль, вершкове масло
  • - batterVolume - об'єм тіста - 1 літр
  • - servingSize - розмір порції - 100гр
  • - toppingChoice - вибір начинки - малина, полуниця, вишня, персик
  • - remainingBatter - залишок тіста - null
    • -
  • patelniaOilLevel - рівень олії в пательні - 3%
    • 1. Створіть файл `hw3.js` 2. Оголосіть в ньому вищенаведені змінні та ініціалізуйте їх значеннями різних типів даних.
